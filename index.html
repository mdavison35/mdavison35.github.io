Professional Self-Assessment

Completing this capstone project, and going through the computer science course as a whole, has showcased my adaptability and determination. I gained experience working in a collaborative environment, using git, and managing merging/conflicts during the class CS310 - Collaboration and Team Projects. I learned valuable skills for communicating with stakeholders in the CS319 - UI/UX Design and Development where I solicited and implemented user feedback. In the classes CS305 - Software Security, and CS405 - Secure Coding I gained valuable experience identifying and mitigating common software vulnerabilities. In the class CS465 - Full Stack Development I I built a full stack web application using the MEAN stack, showing my software engineering ability as well as implementing a MongoDB Database.

Moreover, in the class CS300 - Data Structures and Algorithms: Analysis and Design I gained valuable experience implementing data structures and algorithms, including managing linked lists and pointers.

For this capstone project, I implemented a full stack web application using the Elixir programming language and the Phoenix framework. The application allows a user to login using a 3rd party auth library, and then allows them to upload and view quotes. The upload can be done either with a .csv file or using a form to add individual items. The purpose of this is to allow and Underwriter in an insurance context, to view their outstanding quotes. In doing this, it allows the user to view quotes by broker or brokerage, as well as styling the text size and color to indicate the premium size of the quote.
